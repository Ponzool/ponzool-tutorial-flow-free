# Changelog

## 0.1.0-dev — 2026-07-15

Development preview; not yet licensed for public distribution.

### Added

- Resource-based tutorial pages, definitions, and Catalog.
- Event-triggered popup with multiple pages and keyboard-accessible controls.
- Priority queue with duplicate suppression.
- Pause acquisition and restoration across queued tutorials.
- Always, session, and global display scopes.
- ConfigFile persistence and versioned state export/import.
- Catalog validation, public signals, and history reset APIs.
- Inspectable enemy, item, and simultaneous-event demo.
- Automated runtime and demo verification for Godot 4.7.
